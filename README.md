# Inferior

### CST 338 — Project 2 | Group 7

> A social platform with a twist — not every user gets the same experience.

---

## About

Inferior is a Java desktop app where users can sign in, browse posts from others, and publish their own. The catch: depending on who you are, you might encounter a different interface entirely when you log in.

## Features

- User login & sign up
- Public post feed
- Direct messaging
- Role-based interface — some users see something different on sign in

## Running the Project

1. Clone the repo
   ```bash
   git clone https://github.com/fsotelo-dev/Inferior.git
   ```
2. Open in your IDE (IntelliJ recommended)
3. Run `Main.java`
